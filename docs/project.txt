#### Orhon Framework ####
#########################

- written in pure Orhon, except for bindings like sdl3
- collection of libs that work well with each other
- very modular
- good coding sytle, clean
- no hacks or workarounds
- cross platform
- should support both imediate and retained mode for GUI
- lean and fast
- this project is also a testbed for orhon
